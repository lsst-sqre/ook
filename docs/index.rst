###
Ook
###

Ook is the documentation librarian service for Rubin Observatory, created by the SQuaRE team.

Ook indexes documentation sources, parses and reduces their content, and synchronizes the data with Algolia.
Algolia is the search backend to our documentation portal, https://www.lsst.io.
